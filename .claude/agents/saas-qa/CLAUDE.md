# 하네스: SaaS QA Tester

**목표:** 완성된 Next.js SaaS 앱을 8단계로 자동 테스트하고 4개 카테고리로 10점 만점 채점한다.

## 에이전트 팀

| 에이전트 | 역할 |
|---------|------|
| saas-qa | 완성된 SaaS 앱 블랙박스 테스트 및 채점 (8단계, Playwright + curl) |

## 스킬

| 스킬 | 용도 | 사용 에이전트 |
|------|------|-------------|
| saas-qa-tester | 8단계 자동 테스트 + 4카테고리 채점 + Playwright 브라우저 검증 | saas-qa |

## 실행 규칙

- SaaS 앱 QA/테스트/검증/채점/피드백/점수/평가/리뷰 요청 시 `saas-qa-tester` 스킬을 통해 `saas-qa` 에이전트 단독으로 처리
- 다시 테스트, 재검증, 업데이트 후 재채점도 포함
- 단독 실행 (다른 에이전트와 협업 없음)

## 채점 루브릭 (10점 만점)

| 카테고리 | 비중 | 항목 |
|---------|------|------|
| A. 기능 완성도 | 30% | spec 동작, API shape, stub 없음, 빌드, 마이그레이션 |
| B. 디자인 품질 | 30% | AI 패턴 회피, site.ts 반영, shadcn/ui 활용, 색상/다크 |
| C. 사용성 | 20% | 온보딩, 빈 상태, 에러 피드백, 로딩, 404 |
| D. 코드 품질 | 20% | Feature 규약, any 금지, lint, 에러 핸들링 |

## 자동 실패 조건

- pnpm build 실패
- 서버 기동 실패
- spec 기능 50% 이상 미동작
- 블로킹 버그 (앱 전체 접근 불가)

## 산출물

- `feedback.md` — PASS/FAIL + 점수 + 상세 피드백 + 수정 우선순위
- `screenshots/` — 주요 페이지 데스크톱/모바일 스크린샷
- `###RESULT###...###END###` 블록 — 파싱용 JSON
